Introduction to Web Services : A Web Services is can be defined by following ways:
            * It is a client-server application or application component for communication.
            * The method of communication between two devices over the network.
            * It is a software system for the interoperable machine to machine communication.
            * It is collection of standards or protocols for exchanging information between two 
            devices or application.
        There are mainly two types of web services :
        * SOAP web Services
        * RESTful web services

SOAP : SOAP stands for Simple Object Access Protocol.It is XML-based protocol for accessing web 
        services.
    SOAP is a W3C recommendation for communication between two application.
    SOAP is XML based protocol. It is a platform independent and language independent.By using SOAP,
    you will be able to interact with other programming language applications.


REST : REST stands for Representational State Transfer.
        REST is an architectural style not a protocol.
